# Income-Planner
A modular Python console project implementing an Income Planner with object-oriented design, file I/O, and structured data management.

📌 About

Income Planner helps users track their finances from the command line — add income/expenses, view category-wise breakdowns, calculate savings, and save data to a file.

✨ Features

- Add / View Income
-  Add / View Expenses
-  Calculate Savings (Income − Expenses)
-  Category-wise Tracking
-  Monthly Summary
-  File Save / Load

🛠️ Tech Stack

- Python 3.x
- Console / Terminal (menu-driven)
- Exception Handling & File Handling
